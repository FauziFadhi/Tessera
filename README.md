## How to Run

1. make sure you have nodejs version with minimal `v20.8.1`
2. install all dependencies.
```shell
  npm install
```
3. run database migrations
```shell
  npm run migrate
```
4. run this command to run the project.
```shell
  npm run start:dev
```
